## Remaining Items

### `fstar_support.c` shims (trusted, documented)

`fstar_support.c` provides `FStar_SizeT_v` and `FStar_SizeT_uint_to_t` because the Pulse implementation uses `nat` for array indices (lo, hi, p) but `SizeT.t` for lengths. The casts are safe (all values are small non-negative array indices). A future improvement would extract `FStar.SizeT` directly to C via KaRaMeL.

### `prims.c` int32_t/krml_checked_int_t coupling

The extracted code calls `Prims_op_LessThan` etc. with `krml_checked_int_t` args but `prims.c` uses `int32_t`. Works because `krml_checked_int_t` is typedef'd to `int32_t`. This is a KaRaMeL-level concern, not chapter-specific.
